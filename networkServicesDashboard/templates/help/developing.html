Prerequisites

  • Knowledge of Python and Flask
  • Flask is a super easy framework to pick up in a very short time. It's similar to a lightweight Django.

What needs to be done?

I wrote a lot of this in what I would probably consider sub-optimal conditions.

  • The DMZ portion was written when I was brand new to the language and framework, so things might not have been done in the best way
  • The ITaaC portion was written in about 2 weeks as I was leaving Cisco and had another project that was higher priority.
  • As a result, I cut my fair share of corners. It would be greate if someone could fix up some of the things I couldn't do properly.

How can you help?

This project is hosted on GitHub and the link is in the footer. You should go and contribute! The name of the person maintaining the application can be found at the bottom. If you want to actually take this over just contact them and grab or fork the repo. You can grab ownership of the host as well, no-one is really in charge of this. Make your updates and ssh in and pull them, just clear it with the Network Services SMO team first. Contacts for that might be: (Joe Silver, Jerry Applegate, Samidha Parab).
