Introduction

Welcome to the Network Services Dashboard. This dashboard aims to provide a simple, smooth and fast engagement process for supported services.

These currently include:

  • DMZ as a Service (DMZaaS)
  • IT as a Carrier (ITaaC)

DMZaaS

This part of the application tracks all of the DMZaaS engagements and clients. This is accomplished using three tables.

  1. In Progress Clients
  2. In Service Clients
  3. Widthdrawn Clients
Each of these are slightly different but the main principle remains the same.

There are a number of columns providing information:

Column Details
Lab ID The ID of the Lab associated with this deployment as shown in the LabReg tool. It is an InfoSec requirement that all deployments have an associated lab
Subscriber The name associated with this deployement.
Location Where the lab containing the DMZ is physically located. (Note: Not the DC the DMZ is conencting to)
Status The status that the deployment is currently in (Initial Engagement, Security Review, Implementation)
Assignee The team member this deployment is currently assigned to
Date Updated The Date/Time that this client profile was last updated
Time in status The number of days that the deployment has been in status it is currently in
Monthly Cross Charge The charge the owning department is billed for the service per month. Only present for the In Service view.
